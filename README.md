# Gmail Compose QA Assignment

This repository contains test cases for the Gmail Compose functionality.

## Files Included

1. Gmail_Compose_TestCases.xlsx
   - Contains traditional/manual QA test cases
   - Includes both positive and negative scenarios

2. BDD_Gmail_TestCases.xlsx
   - Contains BDD-style test scenarios
   - Written using Given-When-Then format

## Scope Covered

- Compose email functionality
- Subject validation
- Email body validation
- Recipient validation
- Send email workflow
- Error handling scenarios

## Testing Type

- Manual Testing
- Functional Testing
- Positive Testing
- Negative Testing
- BDD Testing
